## Getting Started

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/VishwanathDalawai/Memory-Game.git
   ```
2. Install dependencies
   ```sh
   npm i
   ```
3. Run dev server
   ```sh
   npm run dev
   ```
